真值表与必要性分析

从校准后的数据到分析的核心工具

1. 什么是真值表？

校准完成后，下一步就是构建真值表（truth table）。真值表是 fsQCA 的核心分析工具，它将数据排列成所有可能的条件组合。

真值表的基本结构

如果有 k 个条件，真值表就有 2k 行，每行代表一种逻辑上可能的条件组合。

例如：你有 3 个条件（A, B, C），就有 23 = 8 种可能的组合。

每一行包含：条件组合、案例数、一致性分数、结果编码。

3. 两个关键阈值

频率阈值（Frequency Threshold）

  • 小样本（10-30）：频率 ≥ 1
  • 中等样本（30-100）：频率 ≥ 2 或 3
  • 大样本（100+）：频率 ≥ 5

一致性阈值（Consistency Threshold）

  • ≥ 0.80：基本可接受
  • ≥ 0.85：良好
  • ≥ 0.90：优秀（推荐）

4. 必要性分析

构建真值表前，惯例是先做必要性分析。

必要性分析问的是：

结果发生时，这个条件是否总是存在？
即：Y ⊆ X（结果 Y 是条件 X 的子集）

必要性一致性 = ∑min(Xi, Yi) / ∑Yi
两个关键指标

一致性：通常要求 ≥ 0.90 才认为存在必要关系。

覆盖度：该必要条件在多大程度上解释了结果。

常见误区：不要混淆必要性和充分性！
  • 必要性：结果发生 → 条件存在
  • 充分性：条件存在 → 结果发生

5. 必要性分析示例

创新研究中的必要性分析结果
条件一致性覆盖度是否为必要条件？
高 R&D (R)0.920.78✅ 是
高教育 (E)0.850.71❌ 否
强知识产权 (P)0.880.65❌ 否
¬R（非高R&D）0.350.42❌ 否

结论：高 R&D 投入是高创新产出的必要条件——没有高 R&D，很难有高创新产出。

6. 检查理解

✍ 测验

必要性分析与充分性分析的区别是什么？

要点回顾

  1. 真值表有 2k 行，每行代表一个条件组合
  2. 两个关键阈值：频率阈值和一致性阈值
  3. 必要性分析在真值表分析之前进行
  4. 必要性一致性通常要求 ≥ 0.90
  5. 必要条件和充分条件不同，不能混淆
