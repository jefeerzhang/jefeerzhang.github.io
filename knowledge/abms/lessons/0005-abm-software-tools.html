NetLogo：最快的入门方式

NetLogo 是 Northwestern University 开发的 ABM 平台，专为教育和研究设计。

NetLogo 快速上手

  1. 下载安装（免费）
  2. 打开 Models Library → 选择 Social Science → Segregation
  3. 点击 Setup，再点击 Go——你就能看到 Schelling 模型的实时运行！
  4. 调整 %similar-wanted 滑块，观察隔离程度如何变化

NetLogo 的优势是即时可视化：写几行代码就能看到 Agent 在屏幕上移动。缺点是对于复杂模型，编程能力有限。

Mesa：Python ABM 框架

Mesa 是 Python 生态中最流行的 ABM 框架。它让你用 Python 代码定义 Agent、环境和调度器。

Mesa 的核心概念映射

ABM 概念Mesa 中的对应
Agentmesa.Agent 的子类
Environmentmesa.Model 的子类 + mesa.space
Schedulermesa.time（控制 Agent 行动顺序）
Data Collectionmesa.DataCollector

用 Mesa 实现 Schelling 模型（核心代码）

import mesa

class SchellingAgent(mesa.Agent):
    def __init__(self, unique_id, model, agent_type):
        super().__init__(unique_id, model)
        self.type = agent_type  # 0 或 1

    def step(self):
        # 统计邻居中同类的比例
        similar = 0
        total = 0
        for neighbor in self.model.grid.iter_neighbors(
            self.pos, moore=True):
            total += 1
            if neighbor.type == self.type:
                similar += 1

        # 如果不满意（同类比例低于阈值），随机搬家
        if similar / total < self.model.homophily:
            self.model.grid.move_to_empty(self)


class SchellingModel(mesa.Model):
    def __init__(self, width=20, height=20, density=0.8,
                 homophily=0.3, agent_type_ratio=0.5):
        super().__init__()
        self.homophily = homophily
        self.grid = mesa.space.SingleGrid(width, height, torus=True)
        self.schedule = mesa.time.RandomActivation(self)

        # 放置 Agent
        for _, pos in self.grid.coord_iter():
            if self.random.random() < density:
                a_type = 1 if self.random.random() < agent_type_ratio else 0
                agent = SchellingAgent(
                    self.next_id(), self, a_type)
                self.grid.place_agent(agent, pos)
                self.schedule.add(agent)

    def step(self):
        self.schedule.step()

运行实验

# 运行 100 步
model = SchellingModel(homophily=0.3)
for _ in range(100):
    model.step()

编程建议

给不同背景的新人建议

  • 完全零基础：先玩 NetLogo，理解 ABM 逻辑后再学编程
  • 会一点 Python：直接用 Mesa，边学边做
  • 有编程经验：Mesa 或 Agents.jl，取决于你的语言偏好
  • 想做大模型：Repast 或 C++，注意计算效率

测验

Q1: 在 Mesa 中，Agent 的行为逻辑写在哪个方法里？

__init__() 方法 step() 方法 run() 方法 observe() 方法

Q2: 在 Schelling 的 Mesa 实现中，homophily=0.3 表示什么？

30% 的 Agent 是同一类型 Agent 要求至少 30% 的邻居是同类才会满意 模型运行 30 步后停止 30% 的格子是空的
