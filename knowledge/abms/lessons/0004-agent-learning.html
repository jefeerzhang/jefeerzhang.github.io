Lesson 04 · 学习机制

Agent 如何学习？从简单到深度

前三课建立了 ABM 的概念框架。现在来回答一个关键问题：Agent 如何适应不断变化的环境？

如果 Agent 只会重复固定行为，模型就太简单了。真正有趣的 ABM 模型中，Agent 会学习——根据过去的经验调整自己的行为。

学习的层次：从简单到深度

Tesfatsion (2024) 将 ABM 中常见的学习算法分为几个层次，从最简单的到最复杂的：

层次 1：强化学习 (Reinforcement Learning)

强化学习的核心思想

Agent 维护一组可选行为，每个行为有一个"吸引力分数"。每次选择某个行为后：

  • 如果结果好（得到高回报）→ 增加该行为的吸引力
  • 如果结果差 → 降低该行为的吸引力

下次选择时，吸引力高的行为被选中的概率更大。

生活类比：你去过几家餐厅吃饭。哪家好吃，下次就更倾向于去哪家。你不需要理解厨师的配方——你只是根据过去的体验来调整偏好。

这是最简单的学习形式：Agent 只需要记住"哪个行为过去给过我好结果"。

层次 2：信念学习 (Belief Learning)

Agent 不仅跟踪自己行为的结果，还维护一个关于环境或其他 Agent 行为的"信念模型"。

例子：在一个重复博弈中，你观察到对手过去 70% 的时间选择了"合作"，于是你"相信"对手合作的概率大约是 70%，并据此选择自己的最优策略。

信念学习比强化学习更"聪明"，因为 Agent 不仅知道"什么行为好用"，还试图理解"为什么好用"。

层次 3：前瞻学习 (Anticipatory / Forward-Looking Learning)

Agent 不仅回顾过去，还预测未来。它有一个关于环境的内部模型，可以在行动前"想象"可能的结果。

例子：一个投资者预测"如果利率上升，债券价格会下跌"，于是在利率上升前就卖出了债券。这需要 Agent 拥有某种"世界模型"。

层次 4：进化学习 (Evolutionary Learning)

受生物进化启发。一个"种群"中的策略经历选择、交叉、变异的过程：

  1. 选择：表现好的策略被保留，差的被淘汰
  2. 交叉：两个好策略的"片段"组合产生新策略
  3. 变异：策略的某些部分随机改变，引入新探索

Holland (1992) 的遗传算法（Genetic Algorithm）就是最经典的例子。

层次 5：连接主义学习 (Connectionist / Neural Network Learning)

使用神经网络作为 Agent 的"大脑"。Agent 通过与环境的交互训练神经网络，使输入（环境状态）到输出（行为选择）的映射不断优化。

这是目前最"深度"的学习形式，但计算成本也最高。

个体学习 vs 群体学习

Vriend (2000) 指出了一个容易被忽视但极其重要的区别：

学习类型含义例子
个体学习 (Individual)Agent 从自己的经验中学习每个公司独立调整定价策略
群体学习 (Social/Population)Agent 从其他 Agent的经验中学习公司模仿表现最好的竞争对手

为什么这个区别很重要？

Vriend 用一个简单的市场模型证明：同样的遗传算法，用在个体层面和群体层面，会产生截然不同的结果。

个体学习时，每个公司独立进化，策略多样性保持得更好，市场更有竞争性。

群体学习时，所有公司都模仿当前最好的策略，导致策略趋同，市场容易陷入"局部最优"。

合作演化：Axelrod 锦标赛

Robert Axelrod 在 1980 年代组织了两场著名的计算机锦标赛，邀请学者们提交策略来参加重复囚徒困境博弈。结果出人意料：

获胜者是最简单的策略——Tit-for-Tat（以牙还牙）：

  • 第一轮：合作
  • 之后每轮：做对手上一轮做的事

Tit-for-Tat 获胜的原因不是它最"聪明"，而是它同时具备：

  1. 善意：先合作
  2. 报复：被背叛就反击
  3. 宽容：对手回到合作就原谅
  4. 清晰：策略简单到对手能"看懂"

测验

Q1: Agent 根据"哪个行为过去给了好回报"来选择下一步行动，这属于哪种学习？

强化学习 (Reinforcement Learning) 信念学习 (Belief Learning) 前瞻学习 (Forward-Looking Learning) 进化学习 (Evolutionary Learning)

Q2: Vriend (2000) 的关键发现是什么？

遗传算法总是优于强化学习 群体学习总是优于个体学习 同一算法在个体层面和群体层面使用会产生不同结果 Agent 不需要学习也能达到最优

本课小结

  • ABM 中的 Agent 可以使用从简单到复杂的多种学习算法
  • 五个层次：强化学习 → 信念学习 → 前瞻学习 → 进化学习 → 连接主义学习
  • 个体学习与群体学习的区别至关重要，会导致截然不同的宏观结果
  • Axelrod 锦标赛证明了：简单策略（Tit-for-Tat）在重复博弈中可以胜过复杂策略
