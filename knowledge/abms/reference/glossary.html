ABMS / ACE 术语表

按字母排序。中文术语后括号内为英文原文，便于查阅原始文献。

A–C

Allocative Efficiency（配置效率） — 市场中互利交易被实现的程度。ZI 交易者实验证明双向拍卖制度能产生接近 100% 的配置效率。

Artificial Stock Market（人工股票市场） — Santa Fe Institute 的经典 ABM，用进化学习的投资者模拟股票市场动态。

Agent（主体） — ABM 中的基本单元。可以是人、组织、自然现象等。每个 Agent 有局部状态和行为规则。

Agent-Based Computational Economics, ACE（基于主体的计算经济学） — c-ABM 在经济学领域的特化。视经济为开放式、本地建构的序贯博弈。

Agent-Based Modeling, ABM（基于主体的建模） — 从个体行为规则出发，通过计算机模拟观察宏观涌现现象的方法论。

Anticipatory Learning（前瞻学习） — Agent 使用内部模型预测未来，据此做决策。比强化学习和信念学习更复杂。

B

Belief Learning（信念学习） — Agent 维护关于环境或对手行为的概率模型，据此选择最优策略。

Bounded Rationality（有限理性） — Herbert Simon 提出。Agent 不具备完全信息和无限计算能力，只能在有限信息下做出"足够好"的决策。

C

Cellular Automata（元胞自动机） — 一种规则驱动的网格模型，每个格子的状态取决于邻居状态。生命游戏是经典例子。

Completely Agent-Based Modeling, c-ABM（完全基于主体的建模） — Tesfatsion 提出的建模框架，包含七条建模原则 (MP1–MP7)，要求所有动态结果完全由 Agent 局部交互产生。

Connectionist Learning（连接主义学习） — 使用神经网络作为 Agent 的决策引擎。

D–F

Deduction（演绎） — 从一般假设推导具体定理的科学方法。

Equifinality（同果异因） — 多个不同的微观机制产生相同的宏观结果。是 ABM 验证的核心挑战。

Deduction（演绎） — 从一般假设推导具体定理的科学方法。

Emergence（涌现） — 系统的宏观属性无法通过分析单个 Agent 来预测，而是从大量 Agent 交互中"冒出"。

Environment（环境） — Agent 生活和交互的空间。可以是网格、网络、市场等。

Evolutionary Learning（进化学习） — 受生物进化启发的学习算法：选择、交叉、变异。

G–I

Game of Life（生命游戏） — Conway (1970) 发明的元胞自动机，只有三条规则却能产生极复杂的行为。

Genetic Algorithm（遗传算法） — Holland (1992) 提出。模拟自然选择过程的搜索和优化算法。

Heuristic（启发式） — ABM 四大研究目标之一：发现反直觉的因果机制和理论洞察。

Induction（归纳） — 从数据中发现模式的科学方法。

Individual Learning（个体学习） — Agent 只从自己的经验中学习。与群体学习 (Social Learning) 相对。

L–M

Locally-Constructive Sequential Game（本地建构序贯博弈） — ACE 对经济系统的核心定义：开放式、本地信息、序贯决策。

Modeler as Petri Dish Experimenter（建模者如培养皿实验员） — MP7：建模者设定初始条件、观察结果，不在模拟过程中干预 Agent。

N

Network Topology（网络拓扑） — Agent 之间连接关系的结构。常见类型：规则网络、随机网络、小世界网络。网络结构深刻影响宏观结果。

Normative（规范性） — ABM 四大研究目标之一：评估制度/政策设计的好坏。

Path Dependence（路径依赖） — 系统的当前状态依赖于历史路径。初始条件的不同可能导致完全不同的最终结果。

Power Law（幂律） — 频率与规模成反比的分布。战争规模、城市规模、词频等都近似服从幂律。

R–S

Reinforcement Learning（强化学习） — 最简单的学习形式：Agent 根据过去行为的回报来调整选择概率。

Schelling Segregation Model（Schelling 隔离模型） — Schelling (1971) 的经典 ABM，展示微观温和偏好如何导致宏观极端隔离。

Simulation（模拟） — 从严格假设出发、用计算机生成数据的第三种科学方法。ABM 属于此类。

Social Learning（群体学习/社会学习） — Agent 从其他 Agent 的经验中学习，如模仿。

Standing Ovation Problem（起立鼓掌问题） — Miller & Page (2004) 提出的 ABM 教学范例，阐释涌现和集体行为。

T

Tit-for-Tat（以牙还牙） — Axelrod 锦标赛的获胜策略：先合作，之后模仿对手上一轮的行为。

V–Z

Validation（验证） — 确认模型结果可信的过程。分四类：输入验证、过程验证、描述性验证、预测性验证。

Walrasian Auctioneer（瓦尔拉斯拍卖人） — 传统经济学中假设的全局价格调整机制。c-ABM 明确排斥这种“上帝之手”。

Zero Intelligence Trader（零智能交易者） — Gode & Sunder (1993)：随机出价但遵守预算约束的交易者。证明市场结构本身能产生高效率。
