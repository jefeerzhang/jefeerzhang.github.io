c-ABM 七条建模原则速查表

来源：Tesfatsion, L. "Agent-Based Computational Economics: Overview and Brief History" (2022). 配套 Lesson 03 使用。

编号 原则名称 核心要求 违反示例
MP1 主体定义
Agent Definition		 明确定义每个 Agent 及其局部状态 模糊地说"市场参与者"而不具体定义
MP2 主体范围
Agent Scope		 所有动态过程都由 Agent 产生 引入全局拍卖人、社会规划者
MP3 主体局部建构性
Local Constructivity		 Agent 只能使用自己的局部信息 让 Agent 直接获取全局状态
MP4 主体自主性
Agent Autonomy		 每个 Agent 独立决策 外部强制指定 Agent 行为
MP5 系统建构性
System Constructivity		 t+1 时刻状态完全由 t 时刻决定 "跳跃到均衡"而不走模拟过程
MP6 系统历史性
System Historicity		 系统状态依赖历史路径 假设系统总能达到唯一均衡
MP7 建模者角色
Petri Dish Experimenter		 设定条件、观察、不干预 模拟中途手动修改 Agent 行为

核心思想一句话

一切动态结果必须完全从 Agent 的局部交互中涌现——没有"上帝之手"。

ACE = c-ABM + 经济学

ACE 把 c-ABM 的七条原则应用到经济系统。核心观点：

  • 经济 = 开放式（无预设均衡）的 本地建构（基于局部信息）序贯博弈（轮流行动）
  • Agent 是异质的、有限理性的、可学习的
  • 均衡是涌现的结果，不是预设的前提
  • 历史很重要——路径依赖是核心特征

自查清单：你的模型符合 c-ABM 吗？

  1. ☐ 我能列出模型中所有 Agent 类型及其状态变量吗？（MP1）
  2. ☐ 模型中有没有"外部协调机制"偷偷替代了 Agent 的交互？（MP2）
  3. ☐ 每个 Agent 的决策是否只依赖它能看到的信息？（MP3）
  4. ☐ 有没有什么东西在强制 Agent 做特定选择？（MP4）
  5. ☐ 系统的每个时间步是否都经过完整的模拟过程？（MP5）
  6. ☐ 不同的初始条件/随机种子会导致不同结果吗？（MP6）
  7. ☐ 模拟过程中我有没有手动修改任何东西？（MP7）
