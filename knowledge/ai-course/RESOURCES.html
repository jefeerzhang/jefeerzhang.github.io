推荐学习资源

高质量的知识资源、社区和实践平台，用于在课程之外拓展学习。

📚 知识资源

资源说明
Microsoft AI-For-Beginners 课程主页 12 周 24 课的完整 AI 入门课程，包含讲义、Jupyter Notebook、Quiz 和 Lab。本课程的底层来源。
AI-For-Beginners 中文版总览 简体中文总览，包含课程路线图、前置要求和每节课对应 Notebook 的链接。学习前通读。
AI-For-Beginners 中文讲义目录 中文讲义和可执行 Notebook。每节课先读 README，再运行对应 Notebook。
《动手学深度学习》- 李沐等 开源中文深度学习教材，配套视频和代码。用于补充对 PyTorch/TensorFlow 实现细节的理解。
_Deep Learning_ - Goodfellow 等 经典"花书"，系统覆盖数学基础与深度学习理论。用于查阅概念、符号和推导。
吴恩达《机器学习》- Coursera 经典入门课程，适合补充机器学习基础概念（监督学习、无监督学习、梯度下降等）。
3Blue1Brown《Neural Networks》- B站 可视化讲解神经网络、反向传播与梯度下降。用于在动手前建立直观理解。
"A recipe for training neural networks" - Karpathy 实战调参指南。用于在训练模型遇到 loss 不下降、过拟合等问题时参考。
PyTorch 官方教程 框架官方教程，用于课程中 PyTorch 相关 Notebook 的配套查阅。
TensorFlow 官方教程 框架官方教程，用于课程中 TensorFlow 相关 Notebook 的配套查阅。

🌐 社区与实践平台

社区说明
AI-For-Beginners GitHub Discussions 官方课程讨论区。用于提问课程具体内容和 Notebook 报错。
Stack Overflow - machine-learning / pytorch / tensorflow 高信噪比的编程与模型问题社区。用于排查具体代码错误。
知乎 AI / 机器学习话题 中文讨论社区。用于了解国内学习者的常见困惑和案例分享。
PaddlePaddle 飞桨开源社区 百度开源深度学习平台，提供中文文档、课程和竞赛。
ModelScope 魔搭社区 阿里达摩院推出的模型开源社区，中文模型和示例丰富。
极市平台 计算机视觉开发者社区。用于补充 CV 方向的实战案例和比赛。

⚠️ 已知缺口

  • 目前国内学习者缺少一套与 AI-For-Beginners 课程完全配套的"中文注释 + 本地环境常见问题"的辅助材料。本课程将逐步补充这些 gap。
  • 课程中部分 Notebook 链接指向英文原版，需要在使用时核对中文目录下的对应文件。
  • 尚未收录专门针对"AI 入门 + 国内学习路径"的高质量视频课程，后续可补充。
