AI 入门术语表

快速查阅本课程中反复出现的核心概念。建议在学习过程中反复回看。

人工智能 Artificial Intelligence, AI 基础
让计算机表现出需要人类智能才能完成的行为的能力，例如识别图像、理解语言、下棋、推荐商品等。
机器学习 Machine Learning, ML 基础
AI 的一个分支，通过数据而非硬编码规则让计算机学习规律。常见任务：分类、回归、聚类、降维。
深度学习 Deep Learning, DL 基础
机器学习的一种方法，使用多层神经网络（深层网络）自动提取特征，广泛用于图像、语音、NLP。
神经网络 Neural Network 基础
由大量简单计算单元（神经元）按层连接而成的模型。通过调整连接权重来拟合数据。
神经元 Neuron 基础
神经网络的基本单元。接收输入、加权求和、通过激活函数产生输出。
感知器 Perceptron 基础
最简单的神经网络单元，用于二分类。输入加权求和后通过阶跃函数输出 0 或 1。
激活函数 Activation Function 基础
决定神经元是否“激活”并输出信号的函数。常见：ReLU、Sigmoid、Tanh、Softmax。
损失函数 Loss Function 训练
衡量模型预测与真实值之间差距的函数。训练目标就是最小化损失函数的值。
梯度下降 Gradient Descent 训练
通过沿损失函数梯度的反方向逐步调整参数，来寻找最小损失的优化算法。
反向传播 Backpropagation 训练
高效计算神经网络中每个参数梯度的算法，是梯度下降在神经网络中的实现方式。
过拟合 Overfitting 训练
模型在训练数据上表现很好，但在新数据上表现差。通常因为模型太复杂或训练数据太少。
欠拟合 Underfitting 训练
模型过于简单，无法捕捉数据规律，在训练集和测试集上表现都不好。
训练集 / 验证集 / 测试集 Train / Validation / Test Set 训练
训练集用于学习参数；验证集用于调参和选模型；测试集用于最终评估模型泛化能力。
卷积神经网络 Convolutional Neural Network, CNN CV
专为图像数据设计的神经网络，利用卷积层提取局部特征，是计算机视觉的核心模型。
迁移学习 Transfer Learning CV
将在大数据集上预训练好的模型用于新任务，通常只需微调少量参数，就能在少量数据上取得好效果。
自然语言处理 Natural Language Processing, NLP NLP
让计算机理解、生成和处理人类语言的技术。包括文本分类、翻译、问答、情感分析等。
词嵌入 Word Embedding NLP
将单词映射为低维稠密向量，使语义相近的词在向量空间中距离更近。例如 Word2Vec、GloVe。
大语言模型 Large Language Model, LLM NLP
基于 Transformer 架构、在海量文本上训练的大型神经网络。能通过提示完成多种语言任务，如 GPT、BERT。
提示工程 Prompt Engineering NLP
通过设计输入提示（prompt）来引导大语言模型生成更准、更有用的输出的技术。
负责任 AI Responsible AI 伦理
在 AI 系统设计、部署和使用中考虑公平性、透明性、隐私、安全和可问责性的实践。
