视觉

卷积神经网络与计算机视觉

掌握 CNN 架构、迁移学习和图像分类。

本课目标： 理解卷积和池化如何让神经网络高效处理图像；用迁移学习在少量数据上取得好效果。

1. 图像给全连接网络的问题

一张 1000×1000 的彩色图片有 300 万个像素值。如果直接输入全连接网络，参数量会爆炸，而且网络无法利用“相邻像素相关”这一先验知识。

2. 卷积：局部感受野 + 参数共享

卷积（Convolution）： 用一个小窗口（卷积核，kernel）在图片上滑动，提取局部特征，比如边缘、纹理。

参数共享： 同一个卷积核扫过整张图片，不管猫出现在图片左上角还是右下角，都能被同一个检测器识别。

# PyTorch 中一个简单的卷积层
import torch.nn as nn

conv = nn.Conv2d(
    in_channels=3,   # 输入通道：RGB 彩色图
    out_channels=16, # 输出通道：16 个卷积核，提取 16 种特征
    kernel_size=3,   # 卷积核大小 3×3
    padding=1        # 边缘补 1 圈 0，保持输出尺寸不变
)

# 输入：1 张 3 通道 32×32 的图片
# 输出：1 张 16 通道 32×32 的特征图

3. 池化：降维与平移不变性

池化（Pooling）： 把一个小区域压缩成一个值（通常是最大值），降低特征图尺寸，减少计算量，同时让特征对位置变化不那么敏感。

pool = nn.MaxPool2d(kernel_size=2, stride=2)
# 32×32 的特征图 → 16×16

4. CNN 典型结构

[输入图像]
  → Conv2d + ReLU（提取边缘）
  → MaxPool
  → Conv2d + ReLU（提取纹理、部件）
  → MaxPool
  → Flatten（展平）
  → Dense（全连接层）
  → 输出类别概率

5. 迁移学习：站在巨人肩膀上

ImageNet 等大型数据集上预训练好的 CNN 已经学会了通用视觉特征（边缘、形状、纹理）。在新任务上，我们只需要替换最后一层分类器，用少量数据微调，就能取得很好的效果。

import torchvision.models as models

# 加载预训练 ResNet18
model = models.resnet18(pretrained=True)

# 冻结前面所有层参数（只训练最后一层）
for param in model.parameters():
    param.requires_grad = False

# 替换最后一层为新的分类器
num_features = model.fc.in_features
model.fc = nn.Linear(num_features, 10)  # 10 个新类别
