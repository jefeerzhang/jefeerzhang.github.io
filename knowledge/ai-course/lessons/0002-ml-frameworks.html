第 2 课：机器学习基础与框架初探

预计 40 分钟 · 目标：理解监督学习流程，能用 PyTorch 或 TensorFlow 跑通一个线性回归

本课目标： 明白“数据 → 模型 → 损失 → 优化 → 评估”这五步；用 PyTorch 或 TensorFlow 实现一个最简单的线性回归。

1. 机器学习 = 用数据找函数

传统编程：人写规则，输入数据，得到输出。

机器学习：人给数据（输入和期望输出），让计算机自动找到能把输入映射到输出的函数。

传统编程： 规则 + 数据 → 输出 机器学习： 数据 + 输出 → 规则（模型）

2. 监督学习 vs 无监督学习

  • 监督学习（Supervised Learning）： 训练数据有标签。例如：给房价数据（面积 → 价格），训练回归模型；给图片和类别标签，训练分类模型。
  • 无监督学习（Unsupervised Learning）： 数据没有标签。例如：把客户分成不同群体（聚类），或把高维数据降到二维可视化（降维）。
  • 强化学习（Reinforcement Learning）： 智能体在环境中通过奖励学习策略。本课程暂不深入。

3. 通用五步法

  1. 准备数据： 收集、清洗、划分训练集/验证集/测试集。
  2. 选择模型： 线性模型、神经网络、决策树等。
  3. 定义损失函数： 衡量预测与真实值的差距。
  4. 优化参数： 用梯度下降等方法让损失越来越小。
  5. 评估模型： 在未见过的测试数据上检验泛化能力。

4. 线性回归：最小例子

假设房价只和面积有关：价格 = w × 面积 + b。我们不知道 w 和 b，但有一组数据。机器学习就是找出最好的 w 和 b。

PyTorch 版本

import torch # 数据：面积（平米）→ 房价（万元） x = torch.tensor([[50.0], [60.0], [80.0], [100.0]]) y = torch.tensor([[300.0], [360.0], [480.0], [600.0]]) # 模型参数：w 和 b，requires_grad=True 表示需要求梯度 w = torch.randn(1, 1, requires_grad=True) b = torch.randn(1, requires_grad=True) optimizer = torch.optim.SGD([w, b], lr=0.0001) for epoch in range(1000): y_pred = x @ w + b loss = ((y_pred - y) ** 2).mean() # 均方误差 MSE optimizer.zero_grad() loss.backward() optimizer.step() print("w:", w.item(), "b:", b.item()) # 理想情况下 w≈6, b≈0

TensorFlow / Keras 版本

import tensorflow as tf import numpy as np x = np.array([[50], [60], [80], [100]], dtype=np.float32) y = np.array([[300], [360], [480], [600]], dtype=np.float32) model = tf.keras.Sequential([ tf.keras.layers.Dense(1, input_shape=(1,)) # 1 个输入，1 个输出 ]) model.compile(optimizer='sgd', loss='mse') model.fit(x, y, epochs=1000, verbose=0) print("w:", model.layers[0].weights[0].numpy()) print("b:", model.layers[0].weights[1].numpy())
✍️ 动手练习：预测新房价

1. 选择 PyTorch 或 TensorFlow 版本，在 notebook 中运行上面的线性回归代码。

2. 用训练好的模型预测面积为 90 平米时的房价。手动计算应该是多少？模型输出是否接近？

3. 把学习率改成 0.001 或 0.00001，观察 loss 是否还能稳定下降。记录下来：学习率过大或过小会发生什么？
