第 4 课：自建神经网络框架

本课目标： 不依赖 PyTorch/TensorFlow，只用 NumPy 搭建一个模块化神经网络框架。通过亲手实现线性层、激活函数、损失函数、反向传播和参数更新，彻底理解现代框架在背后做了什么。

1. 为什么要"造轮子"？

上一课你用 nn.Module 几行就定义了一个网络，但 loss.backward() 到底做了什么？这一课我们拆开黑盒：自己写前向传播、自己推导梯度、自己更新参数。走通这条路后，任何框架都能心里有数。

2. 损失函数：MSE 与交叉熵

任务类型损失函数输出层
回归均方误差 MSE线性
二分类二元交叉熵Sigmoid
多分类交叉熵Softmax
losses.pyPython
import numpy as np # 均方误差：用于回归任务 def mse_loss(y_pred, y_true): return np.mean((y_pred - y_true) ** 2) # 交叉熵损失：配合 softmax 用于多分类 def cross_entropy_loss(y_pred, y_true): eps = 1e-12 # 防止 log(0) y_pred = np.clip(y_pred, eps, 1.0 - eps) return -np.mean(np.sum(y_true * np.log(y_pred), axis=1))
💡 关键直觉： 交叉熵衡量"预测概率分布"与"真实分布"的差异。模型对正确类别给出高概率时损失就低，给错类别高概率时损失被狠狠惩罚。

3. 反向传播：链式法则

多层网络里，损失对深层参数的梯度要靠链式求导法则一层层往回算：

# 两层网络的梯度传播 ∂ℒ/∂w2 = (∂ℒ/∂σ) · (∂σ/∂z2) · (∂z2/∂w2) ∂ℒ/∂w1 = (∂ℒ/∂σ) · (∂σ/∂z2) · (∂z2/∂α) · (∂α/∂z1) · (∂z1/∂w1)

前向传播把数据从输入"流"到输出算出损失；反向传播把误差信号从输出"流"回输入算出每个参数的梯度。

4. 从零构建：线性层

own_framework.pyPython
class Linear: def __init__(self, in_dim, out_dim): self.W = np.random.randn(in_dim, out_dim) * 0.1 self.b = np.zeros((1, out_dim)) def forward(self, x): self.x = x # 记住输入，反向传播要用 return x @ self.W + self.b def backward(self, dz): self.dW = self.x.T @ dz self.db = np.sum(dz, axis=0, keepdims=True) return dz @ self.W.T # 传给上一层

5. 激活函数与 Softmax

def relu_forward(z): return np.maximum(0, z) def relu_backward(dz, z): return dz * (z > 0) def softmax(z): z = z - np.max(z, axis=1, keepdims=True) # 数值稳定 exp_z = np.exp(z) return exp_z / np.sum(exp_z, axis=1, keepdims=True)
⚠️ 常见陷阱： softmax 实现必须先减最大值，否则 np.exp 容易溢出变成 inf。

6. 完整训练循环

training_loop.pyPython
def train(X, y, l1, l2, epochs=100, lr=0.1, batch_size=16): n = X.shape[0] for epoch in range(epochs): idx = np.random.permutation(n) for start in range(0, n, batch_size): xb = X[idx[start:start+batch_size]] yb = y[idx[start:start+batch_size]] # 前向传播 z1 = l1.forward(xb) h = relu_forward(z1) z2 = l2.forward(h) p = softmax(z2) # 反向传播（softmax+交叉熵的梯度 = p - y） dz2 = (p - yb) / p.shape[0] dh = l2.backward(dz2) dz1 = relu_backward(dh, z1) l1.backward(dz1) # 参数更新（SGD） l1.W -= lr * l1.dW; l1.b -= lr * l1.db l2.W -= lr * l2.dW; l2.b -= lr * l2.db if epoch % 10 == 0: print(f"epoch {epoch} loss={cross_entropy_loss(p, yb):.4f}")
