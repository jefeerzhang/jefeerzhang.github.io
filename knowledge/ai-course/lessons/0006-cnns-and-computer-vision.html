第 6 课：CNN 与计算机视觉

预计 45 分钟 · 目标：理解卷积神经网络的核心思想，能运行并修改一个图像分类实验

本课目标： 理解卷积和池化如何让神经网络高效处理图像；了解典型 CNN 架构；用迁移学习在少量数据上取得好效果。

1. 图像给全连接网络的问题

一张 1000×1000 的彩色图片有 300 万个像素值。如果直接输入全连接网络，参数量会爆炸，而且网络无法利用"相邻像素相关"这一先验知识。

2. 卷积：局部感受野 + 参数共享

卷积（Convolution）： 用一个小窗口（卷积核 kernel）在图片上滑动，提取局部特征（边缘、纹理）。参数共享： 同一个卷积核扫过整张图片，不管猫出现在左上角还是右下角，都能被识别。

conv_layer.pyPython
import torch.nn as nn conv = nn.Conv2d( in_channels=3, # 输入通道：RGB 彩色图 out_channels=16, # 输出通道：16 个卷积核 kernel_size=3, # 卷积核大小 3×3 padding=1 # 边缘补零，保持输出尺寸 ) # 输入: (1, 3, 32, 32) → 输出: (1, 16, 32, 32)

3. 池化：降维与平移不变性

池化（Pooling）： 把一个小区域压缩成一个值（通常取最大值），降低特征图尺寸，减少计算量。

pool = nn.MaxPool2d(kernel_size=2, stride=2) # 32×32 的特征图 → 16×16

4. CNN 典型结构

[输入图像] → Conv2d + ReLU（提取边缘） → MaxPool → Conv2d + ReLU（提取纹理、部件） → MaxPool → Flatten（展平） → Dense（全连接层） → 输出类别概率

5. 经典 CNN 架构演进

架构年份关键创新
LeNet1998最早的 CNN，用于手写数字识别
AlexNet2012ReLU + Dropout + GPU 训练
VGG2014小卷积核 (3×3) 堆叠
ResNet2015残差连接，训练超深网络（152层+）
💡 残差连接（ResNet 的核心）： 让输入"跳过"一层直接加到输出上：y = F(x) + x。这解决了超深网络的梯度消失问题，让训练 100+ 层的网络成为可能。
