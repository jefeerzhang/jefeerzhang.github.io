第 3 课：第一个神经网络

预计 45 分钟 · 目标：理解感知器、多层感知器和反向传播，能修改简单网络结构

本课目标： 从感知器出发，理解神经网络如何通过堆叠层和激活函数解决非线性问题；能运行课程 Notebook 并观察网络结构对结果的影响。

1. 感知器：最简单的神经元

感知器接收若干输入，每个输入乘以一个权重，求和后加上偏置，再通过一个阶跃函数输出 0 或 1。它能解决线性可分问题，比如“与门”和“或门”，但无法解决“异或门”（XOR）。

# 感知器计算：输出 = sign(w1*x1 + w2*x2 + b) # 如果结果 ≥ 0 输出 1，否则输出 0 def perceptron(x1, x2, w1, w2, b): z = w1*x1 + w2*x2 + b return 1 if z >= 0 else 0 # 与门：w1=1, w2=1, b=-1.5 print(perceptron(1, 1, 1, 1, -1.5)) # 1 print(perceptron(1, 0, 1, 1, -1.5)) # 0

2. 为什么需要激活函数？

如果没有激活函数，多层神经网络就等价于一层线性变换。激活函数引入非线性，让网络学习复杂边界。

  • Sigmoid： 输出 0~1，适合二分类输出概率。
  • ReLU： 正数部分保留，负数置 0。计算简单，是隐藏层最常用的激活函数。公式：f(x) = max(0, x)
  • Tanh： 输出 -1~1，数据中心化。早期常用，现在多被 ReLU 取代。
  • Softmax： 把一组数值变成概率分布，适合多分类输出。

3. 多层感知器（MLP）

把多个神经元按层组织：输入层 → 隐藏层 → 输出层。隐藏层越多，网络能拟合的函数越复杂，但也更容易过拟合。

import torch import torch.nn as nn class MLP(nn.Module): def __init__(self): super().__init__() self.layers = nn.Sequential( nn.Linear(2, 4), # 输入 2 维，隐藏层 4 个神经元 nn.ReLU(), nn.Linear(4, 1), # 输出 1 维 nn.Sigmoid() # 输出 0~1 的概率 ) def forward(self, x): return self.layers(x) model = MLP() print(model)

4. 反向传播：一句话理解

反向传播做的事情是：先让数据从输入走到输出（前向传播），计算损失；然后沿着网络从输出往回走，计算每个参数对损失的贡献（梯度），最后按梯度反方向更新参数。

你可以把它理解为：模型通过“试错”学习。每次预测错了，就回头看看是哪个连接权重最该负责，然后微调它。
