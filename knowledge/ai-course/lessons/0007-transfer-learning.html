第 7 课：迁移学习与训练技巧

预计 40 分钟 · 目标：理解迁移学习原理，掌握数据增强与训练优化技巧

本课目标： 理解为什么预训练模型有效；学会特征提取与微调两种迁移学习策略；掌握数据增强、学习率调度和 Dropout 等实用技巧。

1. 为什么迁移学习有效？

在 ImageNet（130 万张图片、1000 类）上预训练的 CNN，浅层学到了通用视觉特征（边缘、纹理、形状），这些特征对几乎所有视觉任务都有用。迁移学习就是复用这些已学好的特征，在新任务上只训练少量参数。

策略做法适用场景
特征提取冻结所有预训练层，只训练新分类器数据量少（<1000 张）
微调（Fine-tuning）解冻部分层，用小学习率继续训练数据量中等
从零训练不使用预训练权重数据量巨大且领域差异大

2. 特征提取：冻结 + 替换分类器

transfer_learning.pyPython
import torchvision.models as models import torch.nn as nn # 加载预训练 ResNet18 model = models.resnet18(pretrained=True) # 冻结所有参数——不计算梯度、不更新 for param in model.parameters(): param.requires_grad = False # 替换最后一层为新分类器（这层会被训练） num_features = model.fc.in_features model.fc = nn.Linear(num_features, 10) # 10 个新类别

3. 微调：解冻部分层

# 先做特征提取训练几个 epoch，然后解冻最后几层微调 for name, param in model.named_parameters(): if "layer4" in name or "fc" in name: param.requires_grad = True # 微调时用更小的学习率 optimizer = torch.optim.Adam( filter(lambda p: p.requires_grad, model.parameters()), lr=1e-5 # 比从零训练小 10-100 倍 )
⚠️ 关键原则： 微调时学习率一定要小（1e-4 到 1e-5），否则会破坏预训练权重。通常先用特征提取训几轮，再解冻微调。

4. 数据增强

数据增强通过对训练图片做随机变换（翻转、裁剪、旋转、颜色抖动），人为增加数据多样性，是防止过拟合最有效的手段之一。

data_augmentation.pyPython
from torchvision import transforms train_transform = transforms.Compose([ transforms.RandomResizedCrop(224), # 随机裁剪+缩放 transforms.RandomHorizontalFlip(), # 随机水平翻转 transforms.ColorJitter(brightness=0.2, # 亮度 contrast=0.2, # 对比度 saturation=0.2), # 饱和度 transforms.ToTensor(), transforms.Normalize(mean=[0.485, 0.456, 0.406], # ImageNet 标准化 std=[0.229, 0.224, 0.225]) ])

5. 训练技巧小结

  • 学习率调度： 训练过程中逐渐降低学习率，让模型在后期更精细地收敛。
  • 批归一化（BatchNorm）： 在每层输出上做标准化，加速训练、允许更大学习率。
  • Dropout： 训练时随机丢弃神经元，防止模型过度依赖某些特征。
  • 混合精度训练： 用 float16 加速训练、节省显存（大模型时有用）。
