第 1 课：AI 全景与课程导学

预计 30 分钟 · 无需代码基础即可开始 · 动手目标：理解 AI 地图并准备好学习环境

本课目标： 用一张地图搞清楚 AI、机器学习、深度学习、神经网络的关系；安装好 Python 环境；运行第一个“Hello AI”程序。

1. 从一张地图开始

人工智能（AI）是最大的圈。机器学习（ML）是 AI 的一种方法：让计算机从数据里找规律，而不是靠人写死规则。深度学习（DL）是 ML 的一种方法，用多层神经网络自动学习特征。神经网络（NN）是 DL 的核心模型。

AI ├── 符号 AI / 知识表示（规则、专家系统） ├── 机器学习 ML │ ├── 监督学习：有标签数据（分类、回归） │ ├── 无监督学习：无标签数据（聚类、降维） │ └── 强化学习：奖励驱动的学习 │ └── 深度学习 DL ← 神经网络 NN └── 计算机视觉、自然语言处理、语音识别等应用

为什么神经网络能行？ 因为它可以自动从数据中学习“特征”。传统方法需要人工设计特征（比如识别猫时手动提取边缘、纹理），而深度学习让模型自己发现什么特征对任务最有用。

2. 课程路线

Microsoft 原版课程有 24 课，我们将其浓缩为 6 节短课，每节课聚焦一个可交付的小目标：

  1. AI 全景与课程导学（本课）：环境准备 + 概念地图
  2. 机器学习基础与框架初探：线性回归、损失函数、梯度下降 + PyTorch/TensorFlow 对比
  3. 第一个神经网络：感知器、多层感知器、激活函数、反向传播
  4. 卷积神经网络与计算机视觉：CNN、迁移学习、图像分类
  5. 自然语言处理与大语言模型：词嵌入、RNN、Transformer、提示工程
  6. AI 伦理与下一步：负责任 AI、公平性、隐私、课程复习与进阶路径

3. 环境准备

课程中的代码以 Jupyter Notebook 形式提供。推荐两种运行方式：

方式 A：本地 Anaconda（推荐）

# 1. 安装 Miniconda 或 Anaconda # 2. 创建环境 conda create -n ai python=3.11 conda activate ai # 3. 安装基础包 pip install numpy pandas matplotlib scikit-learn jupyter # 4. 安装深度学习框架（二选一即可） pip install torch torchvision # PyTorch pip install tensorflow # TensorFlow # 5. 启动 Jupyter jupyter notebook

方式 B：Google Colab（零安装）

访问 colab.research.google.com，上传课程 Notebook 即可运行。免费版提供 GPU 选项（运行时 → 更改运行时类型 → T4 GPU）。

4. 第一个程序：Hello AI

先不训练模型，只验证环境。运行下面代码，确认 NumPy 和 PyTorch/TensorFlow 已安装。

import numpy as np import torch print("NumPy 版本:", np.__version__) print("PyTorch 版本:", torch.__version__) print("GPU 可用:", torch.cuda.is_available()) # 用 NumPy 做一个简单的矩阵乘法：X @ W + b X = np.array([[1.0, 2.0]]) W = np.array([[0.5], [1.5]]) b = 1.0 y = X @ W + b print("输出:", y) # 输出: [[4.5]]
