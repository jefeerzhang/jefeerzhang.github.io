本课目标： 用一张地图搞清楚 AI、机器学习、深度学习、神经网络的关系；安装好 Python 环境；运行第一个"Hello AI"程序。

1. 从一张地图开始

人工智能（AI）是最大的圈。机器学习（ML）是 AI 的一种方法：让计算机从数据里找规律，而不是靠人写死规则。深度学习（DL）是 ML 的一种方法，用多层神经网络自动学习特征。神经网络（NN）是 DL 的核心模型。

AI ├── 符号 AI / 知识表示（规则、专家系统） ├── 机器学习 ML │ ├── 监督学习：有标签数据（分类、回归） │ ├── 无监督学习：无标签数据（聚类、降维） │ └── 强化学习：奖励驱动的学习 │ └── 深度学习 DL ← 神经网络 NN └── 计算机视觉、自然语言处理、语音识别等应用

为什么神经网络能行？ 因为它可以自动从数据中学习"特征"。传统方法需要人工设计特征（比如识别猫时手动提取边缘、纹理），而深度学习让模型自己发现什么特征对任务最有用。

3. 环境准备

课程中的代码以 Jupyter Notebook 形式提供。推荐两种运行方式：

4. 第一个程序：Hello AI

先不训练模型，只验证环境。运行下面代码，确认 NumPy 和 PyTorch/TensorFlow 已安装。

hello_ai.pyPython
import numpy as np import torch print("NumPy 版本:", np.__version__) print("PyTorch 版本:", torch.__version__) print("GPU 可用:", torch.cuda.is_available()) # 用 NumPy 做一个简单的矩阵乘法：X @ W + b X = np.array([[1.0, 2.0]]) W = np.array([[0.5], [1.5]]) b = 1.0 y = X @ W + b print("输出:", y) # 输出: [[4.5]]
