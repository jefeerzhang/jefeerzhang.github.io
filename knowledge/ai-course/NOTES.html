教学笔记

本课程的设计假设、使用说明和待补充事项。

课程来源

  • 本课程基于当前目录下的 AI-For-Beginners-zh-CN/（Microsoft 官方课程的中文翻译）。
  • 原始课程为 12 周 24 课，本课程将其浓缩为 12 节短课，每节课聚焦一个可执行的小目标。

学习者假设

  • 零基础或仅听过 AI 概念，未系统学习过机器学习/深度学习。
  • 具备基础 Python 编程能力（变量、函数、列表、字典）。
  • 能在本地安装 Anaconda/Miniconda 或愿意使用 Google Colab。
  • 优先阅读中文材料，但能理解代码中的英文关键字和注释。

每节课设计原则

  • 30-50 分钟可完成，包含：知识讲解 + 1 个代码演示 + 1 个动手练习 + 2 个自测问题。
  • 优先使用课程已有 Notebook，在 HTML lesson 中嵌入关键代码片段并链接到完整 Notebook HTML 版本。
  • 中文注释补充关键代码行，解释"为什么这样写"。
  • 每节课末尾列出与下节课的衔接点和推荐阅读。

Notebook 整合方式

本网站采用"内联 + 链接结合"的方式整合 Jupyter Notebook：

  • 每节课页面内联最核心的代码片段（10-30 行），用带语法高亮的代码块展示。
  • 同时在页面中提供 📓 Notebook 链接框，指向用 nbconvert 转换的完整 Notebook HTML 版本。
  • 所有 52 个 Notebook HTML 文件存放在 notebooks/ 目录下，保留原始目录结构。
  • 用户无需安装 Python 环境即可在浏览器中浏览完整代码和输出。

待补充项

  • 需要跟踪学习者进度：第一次开课时尚无 learning-records/，后续根据用户完成情况生成。
  • 如果用户明确学习目的（如备课、转行、做研究、完成项目），应更新课程目标并调整课程顺序。

已知问题

  • AI-For-Beginners-zh-CN 中部分 Notebook 链接指向英文原版 lessons/，引导学习者时应手动指向中文目录下的对应文件。
  • 课程中的 Quiz 目前仍是英文，需要时可在 lessons 中补充中文自测题。
