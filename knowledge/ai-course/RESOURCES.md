# AI 入门学习资源

## Knowledge

- [Microsoft AI-For-Beginners 课程主页](https://github.com/microsoft/AI-For-Beginners)
  12 周 24 课的完整 AI 入门课程，包含讲义、Jupyter Notebook、Quiz 和 Lab。本课程的底层来源。
- [AI-For-Beginners-zh-CN/README.md](AI-For-Beginners-zh-CN/README.md)
  简体中文总览，包含课程路线图、前置要求和每节课对应 Notebook 的链接。学习前通读。
- [AI-For-Beginners-zh-CN/lessons/](AI-For-Beginners-zh-CN/lessons/)
  中文讲义和可执行 Notebook。每节课先读 README，再运行对应 Notebook（PyTorch 或 TensorFlow 版本二选一即可）。
- [Book: 《动手学深度学习》— 李沐、Aston Zhang 等](https://zh.d2l.ai/)
  开源中文深度学习教材，配套视频和代码。用于在课程 Notebook 之外补充对 PyTorch/TensorFlow 实现细节的理解。
- [Book: _Deep Learning_ — Ian Goodfellow, Yoshua Bengio, Aaron Courville](https://www.deeplearningbook.org/)
  经典“花书”，系统覆盖数学基础与深度学习理论。用于在需要时查阅概念、符号和推导。
- [Course: 吴恩达《机器学习》— Coursera / 网易云课堂](https://www.coursera.org/learn/machine-learning)
  经典入门课程，适合补充机器学习基础概念（监督学习、无监督学习、梯度下降等）。
- [Video Series: 3Blue1Brown《Neural Networks》— YouTube / B站搬运](https://www.bilibili.com/video/BV1bx411M7ZL/)
  可视化讲解神经网络、反向传播与梯度下降。用于在动手前建立直观理解。
- [Article: “A recipe for training neural networks” — Andrej Karpathy](https://karpathy.github.io/2019/04/25/recipe/)
  实战调参指南。用于在训练模型遇到 loss 不下降、过拟合等问题时参考。
- [Docs: PyTorch 官方教程](https://pytorch.org/tutorials/)
  框架官方教程，优先用于课程中 PyTorch 相关 Notebook 的配套查阅。
- [Docs: TensorFlow 官方教程](https://www.tensorflow.org/tutorials)
  框架官方教程，优先用于课程中 TensorFlow 相关 Notebook 的配套查阅。

## Wisdom (Communities)

- [Microsoft AI-For-Beginners GitHub Discussions](https://github.com/microsoft/AI-For-Beginners/discussions)
  官方课程讨论区。用于提问课程具体内容和 Notebook 报错。
- [Stack Overflow — 标签 `machine-learning` / `pytorch` / `tensorflow`](https://stackoverflow.com/questions/tagged/machine-learning)
  高信噪比的编程与模型问题社区。用于排查具体代码错误。
- [知乎 AI 话题 / 机器学习话题](https://www.zhihu.com/topic/19559450/hot)
  中文讨论社区。用于了解国内学习者的常见困惑和案例分享。
- [PaddlePaddle 飞桨开源社区](https://www.paddlepaddle.org.cn/)
  百度开源深度学习平台，提供中文文档、课程和竞赛。用于寻找中文实践资源和赛事。
- [ModelScope 魔搭社区](https://www.modelscope.cn/)
  阿里达摩院推出的模型开源社区，中文模型和示例丰富。用于了解国内 AI 应用和模型生态。
- [极市平台](https://www.cvmart.net/)
  计算机视觉开发者社区。用于补充 CV 方向的实战案例和比赛。

## Gaps

- 目前国内学习者缺少一套与 AI-For-Beginners 课程完全配套的“中文注释 + 本地环境常见问题”的辅助材料。本课程将逐步补充这些 gap。
- 课程中部分 Notebook 链接指向英文原版，需要在使用时核对中文目录下的对应文件。
- 尚未收录专门针对“AI 入门 + 国内学习路径”的高质量视频课程，后续可补充。
