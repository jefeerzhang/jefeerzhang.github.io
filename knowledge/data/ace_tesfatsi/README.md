## 一、这个页面是什么

`ace.htm` 不是一本书或一篇论文，而是 **Agent-Based Computational Economics（ACE，基于主体的计算经济学）** 的**资源索引门户**，由 Iowa State University 荣休教授 Leigh Tesfatsion 维护。它系统性地列出 ACE / c-ABM（完全基于主体的建模）领域的：
- 概念框架（7 条建模原则 MP1–MP7）
- 入门材料、方法论文档、教学/自学资源
- 软件工具、各研究领域资源站、期刊与新闻

## 二、核心概念速览（来自索引页正文）

- **c-ABM（完全基于主体的建模）**：把过程建模为"开放式的、由相互作用的主体驱动的动态系统"。主体可小到天气等无认知功能的物理现象，大到机构、社会群体。
- **7 条建模原则 MP1–MP7**：主体定义、主体范围、主体本地建构性、主体自主性、系统建构性、系统历史性、建模者如"培养皿实验员"。
- **ACE = c-ABM 在经济系统的特化**：把真实经济视为"开放式、本地建构的序贯博弈"。
- **四大研究目标**：① 解释经验规律的涌现与持续；② 规范性设计（制度/政策评估）；③ 定性洞察与理论生成；④ 方法/工具推进。

## 五、使用建议

1. 从 `ace_index.html` 离线浏览全部门类；`pdfs/` 是可直接引用的核心文献（9 个）。
2. 新手路线：`intro/abmread.htm` → `pdfs/LearnAlgorithms.LT.pdf` → `teaching/syl308.htm`。
3. 做模型时参考 `method/AModGuide.htm` 与 `method/empvalid.htm`。
4. 找软件/工具看 `software/acecode.htm`；找某领域文献看 `other/` 下对应研究领域页。
5. 想深入某领域，以 `other/` 中的子页为目录，再追其内链的原始论文（深度 3，未下载）。
